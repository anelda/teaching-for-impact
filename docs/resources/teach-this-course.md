# Teaching This Course

In this section we provide useful resources should you want to teach this course.

## Presentations

Slide decks for live sessions. Use these for facilitated sessions. These are designed to be run in 4 hour sessions. They cover the major parts of each part in an interactive way, they are not an exhaustive reference.

**[Part 1 — Designing Training](../presentations/part-1.html){target="_blank"}**
:   Covers Lessons 1–9: from systems thinking and theory of change through to assessment design.

**[Part 2 — From Training Design to OER](../presentations/part-2.html){target="_blank"}**
:   Covers Lessons 1–6: designing for openness, licensing, remixing, and publishing OER.

!!! tip "Keyboard shortcuts"
    - **Arrow keys** or **Space** — navigate slides
    - **Esc** — slide overview
    - **S** — speaker notes (if available)
    - **F** — fullscreen
